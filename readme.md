# Mathas

Multiplayer math game with websockets ([website](https://mathas.azurewebsites.net/))

## First Run

```javaScript
// Install dependencies
npm install

// Start server
npm start
```

## Running Tests

```javaScript
// Unit tests
npm run test

// Automation tests (with Selenium)
npm start // If the server isn't running already
npm run test:auto

```
